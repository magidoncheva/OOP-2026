### Задача 2

Аня много обича пролетта, но не е особен фен на променливото време. Съответно един дъждовен следобед, тя решава да състави метод на систематизация на времето. Тя реализира клас Weather, който описва атмосферните условия в един ден - колко градуса по целзий е температурата навън и сила на вятъра. Съответно, обаче, същестувва и възможността за RainyWeather - когато са възможни валежи и Аня иска да си запише процентната вероятност за тях, SunnyDay - слънчев ден, но Аня няма друга информация за него (просто си отбелязва, че е слънчев), и RandomWeatherDay - времето има две различни състояния и Аня иска да ги запише и двете в един и същи обект.

Да се реализира подходяща йерархия от класове, която повтаря тази на Аня. Използвайте наследяване и композиция на класове, където е необходимо.

### Задача 3

Да се създаде модел на шахматна дъска и шахматни фигури. 

Обозначения на фигурите:
- K - цар
- Q - царица
- R - офицер
- B - топ
- N - кон
- P - пешка
- b - черно
- w - бяло

Обозначения на дъската:
|   |   | a | b | c | d | e | f | g | h |
| - | - | - | - | - | - | - | - | - | - |
| 8 | b | B | N | R | Q | K | R | N | B |
| 7 | b | P | P | P | P | P | P | P | P |
| 6 |   |   |   |   |   |   |   |   |   |
| 5 |   |   |   |   |   |   |   |   |   | 
| 4 |   |   |   |   |   |   |   |   |   | 
| 3 |   |   |   |   |   |   |   |   |   | 
| 2 | w | P | P | P | P | P | P | P | P |
| 1 | w | B | N | R | Q | K | R | N | B |

Да се даде възможност за базово придвижване на всяка една от фигурите:
- цар - едно поле на всички страни
- царица - произволен брой полета на всички страни
- офицер - произволен брой полета по диагонал
- топ - произволен брой полета напред или настрани
- кон - две полета напред или настрани, след това едно поле перпендикулярно
- пешка - едно поле напред
