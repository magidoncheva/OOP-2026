### Задача 1

Да се създаде модел на шахматна дъска и шахматни фигури. 

Обозначения на фигурите:
- K - цар
- Q - царица
- R - офицер
- B - топ
- N - кон
- P - пешка
- b - черно
- w - бяло

Обозначения на дъската:
|   |   | a | b | c | d | e | f | g | h |
| - | - | - | - | - | - | - | - | - | - |
| 8 | b | B | N | R | Q | K | R | N | B |
| 7 | b | P | P | P | P | P | P | P | P |
| 6 |   |   |   |   |   |   |   |   |   |
| 5 |   |   |   |   |   |   |   |   |   | 
| 4 |   |   |   |   |   |   |   |   |   | 
| 3 |   |   |   |   |   |   |   |   |   | 
| 2 | w | P | P | P | P | P | P | P | P |
| 1 | w | B | N | R | Q | K | R | N | B |

Да се даде възможност за базово придвижване на всяка една от фигурите:
- цар - едно поле на всички страни
- царица - произволен брой полета на всички страни
- офицер - произволен брой полета по диагонал
- топ - произволен брой полета напред или настрани
- кон - две полета напред или настрани, след това едно поле перпендикулярно
- пешка - едно поле напред
